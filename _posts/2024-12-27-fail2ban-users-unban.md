Чтобы разблокировать всех пользователей в **Fail2ban**, нужно остановить службу Fail2ban и очистить все правила iptables, созданные этой службой. Вот пошаговая инструкция:

### Шаг 1: Остановка службы Fail2ban

Сначала остановим саму службу Fail2ban, чтобы она больше не блокировала IP-адреса:

```bash
sudo systemctl stop fail2ban
```

### Шаг 2: Удаление правил iptables

Теперь удаляем все правила iptables, добавленные Fail2ban. Для этого выполните команду:

```bash
sudo iptables -F
```

Эта команда очищает все цепочки iptables.

### Шаг 3: Перезапуск службы Fail2ban (опционально)

После очистки правил вы можете снова запустить Fail2ban:

```bash
sudo systemctl start fail2ban
```

### Альтернативный способ через `fail2ban-client`

Также можно использовать утилиту `fail2ban-client` для удаления всех заблокированных IP-адресов без остановки самой службы:

```bash
sudo fail2ban-client unban --all
```

Эта команда разблокирует все IP-адреса, находящиеся под баном.

### Проверка

Для проверки того, что все блокировки были сняты, вы можете посмотреть текущие активные правила iptables:

```bash
sudo iptables -L
```

Теперь все пользователи должны быть разблокированы!
