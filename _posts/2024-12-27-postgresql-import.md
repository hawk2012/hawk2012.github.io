Импорт базы данных в PostgreSQL может осуществляться несколькими способами, но наиболее часто используются следующие методы:

### 1. Использование команды `psql` для импорта дампа SQL

Если у вас есть файл дампа базы данных, созданный командой `pg_dump`, вы можете использовать команду `psql` для восстановления этой базы данных.

#### Шаги:

1. **Создайте новую базу данных** (если она еще не существует):
   ```bash
   createdb имя_новой_базы
   ```

2. **Восстановите данные из файла дампа**:
   ```bash
   psql -U пользователь -d имя_новой_базы -f путь/к/дампу.sql
   ```

   Где:
   - `имя_новой_базы` – название новой базы данных, куда будет восстановлена информация.
   - `пользователь` – имя пользователя PostgreSQL, который имеет права на создание объектов в базе данных.
   - `путь/к/дампу.sql` – полный путь к файлу дампа, который нужно восстановить.

### 2. Использование утилиты `pg_restore`

Если ваш дамп был создан с помощью опции `--format=custom` или `--format=directory`, то для восстановления потребуется использовать утилиту `pg_restore`.

#### Шаги:

1. Убедитесь, что база данных создана:
   ```bash
   createdb имя_новой_базы
   ```

2. Восстановите базу данных из архива:
   ```bash
   pg_restore -U пользователь -d имя_новой_базы путь/к/архиву.dump
   ```

   Здесь:
   - `имя_новой_базы` – новая база данных, которую вы хотите заполнить данными.
   - `пользователь` – пользователь PostgreSQL с правами на создание объектов.
   - `путь/к/архиву.dump` – путь к архивному файлу дампа.

### 3. Импорт через интерфейс администратора (например, pgAdmin)

Если вы используете графический инструмент управления базами данных, такой как pgAdmin, процесс импорта также возможен через GUI-интерфейс.

#### Шаги:

1. Откройте pgAdmin и подключитесь к серверу PostgreSQL.
2. Выберите нужную базу данных в дереве объектов слева.
3. Щелкните правой кнопкой мыши на выбранной базе данных и выберите пункт «Restore» («Восстановление»).
4. В открывшемся окне выберите файл дампа и настройте параметры восстановления.
5. Нажмите кнопку «Restore».
