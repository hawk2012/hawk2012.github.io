<h4>Средство автономной установки СКЗИ КриптоПро</h4>
<p>Первый установщик, который я создавал с помощью Inno Script. Долгое время не поддерживался. Последнее обновление - 30 мая 2024 года.</p>
<p>Программа предназначена для Windows и позволяет производить установку необходимых пакетов из единого интерфейса. Распространяется как исполняемый файл (EXE).</p>

<h4>Скрипт установки драйверов ККМ АТОЛ для Linux</h4>
<p>Данный скрипт предназначен для установки или удаления пакетов программного обеспечения в операционных системах Debian и Ubuntu Linux.</p>
<p>Он предоставляет пользователю меню для выбора действий: установка пакета для архитектуры AMD64 или i386, удаление пакета для одной из этих архитектур или выход из программы.</p>
<p>Скрипт использует функции для установки и удаления пакетов, а также проверяет наличие файлов пакетов перед выполнением операций.</p>
<p><a href="https://github.com/hawk2012/atol_installer" class="btn btn-primary btn-sm" target="_blank">Репозиторий</a></p>

<h4>Bash-скрипт для выполнения различных системных задач</h4>
<p>Этот скрипт предоставляет пользователю меню с различными функциями, которые он может выбрать.</p>
<p>Каждая функция реализована в скрипте как отдельная функция. Пользователь может выбрать одну из функций, ввести необходимые данные (если требуется), и скрипт выполнит соответствующую операцию.</p>
<h5>Функции</h5>
<ul>
<li>проверка всех дисков</li>
<li>получение UUID диска</li>
<li>получение внешнего IP-адреса ПК</li>
<li>создание RAID-массива из указанных дисков</li>
<li>удаление RAID-массива</li>
<li>выполнение трассировки маршрута к указанному IP-адресу или хосту</li>
<li>обновление системы в скрытом режиме</li>
<li>поиск программы в системных репозиториях и установка ее с зависимостями</li>
</ul>
